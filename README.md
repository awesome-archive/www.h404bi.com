## Playground

**Prerequisites:**  
Git, Nodejs, Yarn

### Clone the source:

``` sh
$ git clone https://github.com/h404bi/www.h404bi.com.git
$ cd www.h404bi.com
```

### Run and play:

``` sh
# Bootstrap, install dependencies
$ yarn bootstrap
# Start dev server
$ yarn dev
# Listening 127.0.0.1:3000
```
