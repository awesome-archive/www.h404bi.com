## Playground

**Prerequirements:**  
Git, Nodejs, Yarn

### Clone the source:

``` sh
$ git clone https://github.com/h404bi/h404bi.com.git
$ cd h404bi.com
```

### Run and play:

``` sh
# Install global hexo-cli (Optional)
$ yarn global add hexo-cli
# Install Hexo dependencies and run dev
$ yarn && yarn dev
# Listening 0.0.0.0:4000
```
