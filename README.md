## Playground

**Prerequisites:**  
Git, Nodejs, Yarn

### Clone the source:

``` sh
$ git clone https://github.com/h404bi/www.h404bi.com.git
$ cd www.h404bi.com
```

### Run and play:

``` sh
# Install Hexo dependencies and run dev
$ yarn && yarn dev
# Listening 0.0.0.0:4000
```
